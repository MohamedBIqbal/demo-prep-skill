### Demo Part 1: Happy Path (1:30 - 2:30)

**RUN:** [Your demo command or action]

>

## Pre-Recording Checklist

- [ ] Slides open in browser (full screen)
- [ ] Demo environment ready and tested
- [ ] Backup screenshots/video prepared
- [ ] Screen recorder configured
- [ ] Microphone tested
- [ ] Font sizes increased for visibility
- [ ] Notifications silenced
- [ ] Water nearby
- [ ] This script visible on second screen

---

## If Things Go Wrong

**Demo fails:**
> "Let me show you from the backup."
> → Switch to screenshot or recorded video
> → Continue with narrative

**Running long:**
- Skip Slide 3 (mention numbers verbally)
- Shorten demo to one example
- Prioritize The Ask

**Running short:**
- Expand demo with second example
- Ask "Questions so far?" after Results
